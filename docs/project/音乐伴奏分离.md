## 
```bash

demucs -n htdemucs \
  --two-stems=vocals \
  --shifts 2 \
  --segment 7 \
  --overlap 0.25 \
  --out output_dir \
  video.mp4
  
  
  # 输出 MP3 192kbps
demucs -n htdemucs --two-stems=vocals --mp3 --mp3-bitrate 192 video.mp4

# 输出 FLAC（无损，体积小于 WAV）
demucs -n htdemucs --two-stems=vocals --flac video.mp4



```

| 参数                   | 作用                   |
| -------------------- | -------------------- |
| `-n htdemucs`        | 高质量 HTDemucs 模型      |
| `--two-stems=vocals` | 输出人声 + 伴奏            |
| `--shifts 2`         | 多次预测取平均 → 高质量        |
| `--segment 7`        | Transformer 模型安全最大分段 |
| `--overlap 0.25`     | 分段边缘平滑               |
| `--out output_dir`   | 输出目录                 |
| `video.mp4`          | 输入文件                 |



## 合并

```bash
ffmpeg -i video.mp4 \
       -i output_dir/htdemucs/video/no_vocals.wav \
       -i output_dir/htdemucs/video/vocals.wav \
       -filter_complex "[2:a]volume=0.15[vocals_low];[1:a][vocals_low]amix=inputs=2:normalize=1[a]" \
       -map 0:v -map "[a]" -c:v copy -c:a aac -b:a 192k output_video15.mp4
       

高级优化    normalize=0 平时用上一个
ffmpeg -y \
-i video.mp4 \
-i no_vocals.wav \
-i vocals.wav \
-filter_complex "
[1:a]volume=1.0[bgm];
[2:a]volume=0.15[vocals];
[bgm][vocals]amix=inputs=2:normalize=0[a]
" \
-map 0:v -map "[a]" \
-c:v copy -c:a aac -b:a 192k output_video15.mp4


```
| 部分                                              | 作用                            |
| ----------------------------------------------- | ----------------------------- |
| `[2:a]volume=0.3[vocals_low]`                   | 对人声轨 `vocals.wav` 直接降低音量到 30% |
| `[1:a][vocals_low]amix=inputs=2:normalize=1[a]` | 混合伴奏 + 降低音量人声                 |
| `-map 0:v`                                      | 保留原视频画面                       |
| `-map "[a]"`                                    | 使用混合后的音频                      |
| `-c:v copy`                                     | 视频不重新编码                       |
| `-c:a aac -b:a 192k`                            | 输出 AAC 音频                     |

## 音频操作

```bash
ffmpeg -i output_dir/htdemucs/video/no_vocals.wav \
       -i output_dir/htdemucs/video/vocals.wav \
       -filter_complex "[1:a]volume=0.30[vocals_low];[0:a][vocals_low]amix=inputs=2:normalize=1[a]" \
       -map "[a]" -c:a pcm_s16le  output_audio30.wav

ffmpeg -i output_audio.wav -c copy -t 67 output_audio_cut.wav
```
