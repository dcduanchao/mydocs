# 文件互传

## 本地到 Linux

```bash
# 上传单个文件
scp F:/work/my/study-note/dc-web/target/dc-web.jar root@155.94.170.142:/root/web
scp F:/work/my/study-note/dc-manage/target/dc-manage.jar root@155.94.170.142:/root/manage

# 上传目录里的所有文件
scp -r /f/VUE/wb-web/dist/* root@155.94.170.142:/usr/share/nginx/html/wbweb/
scp -r /f/VUE/wb-manage/dist/* root@155.94.170.142:/usr/share/nginx/html/wbmanage/

# 上传整个目录
scp -r /f/VUE/wb-web/dist root@155.94.170.142:/root/

# 指定 ssh 端口
scp -P 22 F:/work/my/study-note/dc-web/target/dc-web.jar root@155.94.170.142:/root/web/
```

## Linux 到本地

```bash
# 下载单个文件到当前目录
scp root@155.94.170.142:/root/web/dc-web.jar .
scp -v root@155.94.170.142:/root/dump.sql dump.sql

# 下载到指定目录
scp root@155.94.170.142:/root/manage/dc-manage.jar F:/work/my/study-note/

# 下载整个目录
scp -r root@155.94.170.142:/usr/share/nginx/html/wbweb/ F:/work/my/

# 指定 ssh 端口
scp -P 22 root@155.94.170.142:/root/web/dc-web.jar .
```
